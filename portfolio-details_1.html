About this Project

This is the Homepage of the NSBM SPORTS website. The name and logo of this website are at the top of the homepage. After clicking on the logo, the user will be redirected to the official website of NSBM Green University. This homepage has a navigation bar. That navigation bar is included on every page. Then a slider is added. The user can then view recent sports clubs. And by clicking the see more button the user is redirected to the clubs page. After that, the user can ss all sports clubs on one single page. A footer has been added at the end of this homepage. Our NSBM social media links are linked to the social media buttons in that footer. Also, this footer is included at the end of each page
